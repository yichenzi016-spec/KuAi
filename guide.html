从安装到第一个分镜

跟着做，几步就能把零散提示词管起来、把文案变成视频脚本。

快速上手

五个步骤跑通核心流程

1

下载并安装

从下载页获取 V1.1.0 安装包，双击按向导安装。首次打开会创建本机数据文件 软件配置数据(请勿删除).json。

2

配置一个大模型（可选但推荐）

点主界面 🌐 或菜单「工具 → 翻译/润色引擎设置」，选一个引擎（如 DeepSeek / 通义千问）并填入 Key 与模型名。想完全离线，选 Ollama 本地模型即可免 Key。配置后点「测试」确认连通。

3

建第一条提示词

导航到「工作台」→ 点「新建分组」建立分类（如"电商主图"）→ 点「新建卡片」添加提示词。双击卡片打开编辑器，上下两栏分别写中文 / 英文；需要翻译时点 ⇄，需要润色时点 ✨。

4

生成视频提示词

导航到「视频提示词」，左侧选画幅（如 9:16）、密度、模型适配，右侧粘贴文案点「生成」。稍候即可得到可直接贴到即梦 / 可灵 / Seedance 的镜头提示词，点「导出 .docx」留存。

5

做一份智能分镜

导航到「智能分镜」，输入创意点「一键生成」。系统会提取角色 / 场景并生成分镜表；上传角色 / 场景参考图锁定一致性，编辑每镜内容后导出 .xlsx 交给制作或拍摄。

常见场景

按你想做的事，对号入座

📚

只想安静收集提示词

纯本地可用。用工作台建分组、加卡片、打标签；视觉灵感库批量导入参考图并 AI 反推提示词。无需任何 API 配置。

🌐

要把中文提示词翻成英文

编辑器里打开卡片，点 ⇄ 翻译，一键中英互译；也可先 🔮 识别变量 把可变内容参数化，再翻译，批量复用更方便。

🎬

要拍一条短视频

「视频提示词」把文案拆成镜头脚本 → 「智能分镜」生成带景别 / 运镜 / 台词的可编辑分镜表 → 导出 xlsx 直接用于拍摄或喂给 AI 视频工具。

🛡️

怕软件崩、怕丢稿

默认每 2 分钟自动保存 + 50 个历史快照可回退；万一异常退出，下次启动会提示从崩溃前备份恢复。重要项目记得用「导出配置」做备份。

排障小贴士

上手时容易卡住的地方

点生成没反应 / 报错？

多半是 API 没配或 Key 无效。去「工具 → 翻译/润色引擎设置」确认引擎、URL、Key、模型名都填对，并用「测试 / 连通性检测」验证。

翻译按钮是灰的？

翻译依赖已配置的 AI 引擎。先完成第 2 步的引擎配置，按钮即可使用。

分镜里角色"换脸"了？

上传该角色的参考图作为身份资产（控制 identity / wardrobe），软件会跨镜头锁定，避免漂移。详见功能介绍第 4 项。

想换电脑继续用？

菜单「导出配置」存为 .json，到新电脑「导入配置」即可；项目数据文件也可整体复制带走。
