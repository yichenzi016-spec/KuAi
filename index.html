常见问题

下载使用前，先了解这些

软件收费吗？

软件本身免费使用。调用大模型产生的 API 费用按各家服务商标准由你自行承担（本地 Ollama 模型不产生费用）。

必须联网才能用吗？

不需要。提示词管理、分镜表格编辑、视觉灵感库浏览等本地功能离线可用；只有 AI 翻译 / 润色 / 提示词生成 / 图片分析需要联网与 API Key。

我的提示词数据安全吗？

所有项目数据保存在本机 JSON 文件（软件配置数据(请勿删除).json），不上传任何服务器。支持导出备份。

支持哪些视频平台提示词？

输出可对齐即梦、可灵、Seedance 2.0、Veo、Runway、Pika 等，提供多种输出格式与模型适配预设。

没有编程基础能上手吗？

能。软件是图形界面操作，常用功能均有按钮与向导。详见上手教程。
