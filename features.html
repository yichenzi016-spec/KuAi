1

API / 翻译引擎设置

13 家大模型 · 本地 Ollama 免 Key
解决什么：集中配置用哪个大模型、填 Key / 地址 / 模型名，也支持完全离线的本地模型。
  • 主界面点 🌐 或菜单「工具 → 翻译/润色引擎设置」。
  • 选择引擎，填写对应 Key / URL / 模型名（豆包还需 Endpoint ID；百度翻译只需 AppID / Secret）。
  • 选 Ollama 可本地运行，免 API Key，数据不出本机。
  • 支持 DeepSeek、通义千问、Kimi、豆包、智谱、MiniMax、零一万物、讯飞星火、腾讯混元、百度千帆、百度翻译、Ollama、OpenAI 兼容。
13 引擎Ollama 本地统一配置
2

工作台 · 提示词项目管理

卡片 / 分组 · 拖拽 · 搜索 · 撤销重做
解决什么：把零散的提示词集中管理，像文件夹一样分层归类、复用与检索，不再散落在记事本和聊天记录里。
  • 左侧导航点「工作台」，看到卡片网格与分组树。
  • 点「新建卡片」添加单条提示词；点「新建分组」建立文件夹归类。
  • 用鼠标框选多选，右键复制 / 粘贴 / 剪切 / 删除 / 重命名 / 改色。
  • 开启工具栏编辑模式，可拖动重排、改名、改色、改大小、隐藏。
  • 搜索框支持 #标签 与日期区间筛选，快速定位历史提示词。
  • 选中卡片可一键「发送到分镜 / 视频」，跨模块联动。
卡片网格分组树拖拽重排撤销/重做
3

全网采集

一键抓取 · AI 清洗入库 · 提示词反推
解决什么：手动在网上找参考素材太慢？全网采集帮你批量抓取目标平台的内容，自动清洗去噪后 AI 反推成可复用的提示词，直接进入灵感库或提示词库。
  • 导航点「全网采集」，输入目标网址或关键词。
  • 选择采集范围：单页内容 / 搜索结果列表 / 指定平台的素材池。
  • 点「开始采集」，自动抓取文本、图片、元数据，智能过滤广告和无关内容。
  • 抓取结果支持预览 / 筛选 / 编辑，确认后一键发送到视觉灵感库或提示词编辑器。
  • 可选开启 AI 自动反推：对采集到的图片/文案实时调用大模型生成中英文提示词。
  • 支持定时任务：设置采集计划，定期自动更新灵感库内容。
批量抓取AI 反推定时任务多平台
4

视觉灵感库

参考图分类 · AI 反推提示词 · 虚拟滚动
解决什么：把参考图当"灵感图书馆"按风格归档，AI 一键反推出中英文提示词，大图库也不卡。
  • 导航点「视觉灵感」，左侧 9 大分类 + 收件箱，右侧缩略图画廊。
  • 点图片看大图，支持缩放 / 平移；点「AI 分析」用通义千问提取中英文提示词。
  • 复制 cn / en、加标签、收藏，或拖拽归类到分类。
  • 批量导入图片文件 / 文件夹 / 剪贴板，框选多选后批量导出。
  • 一键把图片「发送到编辑器」继续加工。
9 大分类AI 反推拖拽归类虚拟滚动
5

提示词编辑器 · 中英互译 / AI 润色 / 变量

双语维护 · 一键翻译 · 智能变量
解决什么：逐条打磨提示词，中英文双语维护，AI 扩写润色，把重复内容做成可复用变量。
  • 双击卡片打开编辑窗，上下两栏分别是中文 / 英文提示词。
  • 点 ⇄ 翻译 调用当前引擎做中英互译，无需切换软件。
  • 点 ✨ 一键润色，可打开润色设置调整风格（电影感 / 写实 / 简洁等）。
  • 点 🔮 AI 识别变量，自动把人名 / 日期 / 金额等提取为 {{变量}} 填空槽。
  • 也可手动插入 {{}} 槽位，配合预设模板批量套用。
  • 支持 Ctrl+S 保存、一键复制 / 清空。
中英双语AI 润色变量槽模板套用
6

预设模板系统

可复用提示词模板 · 分类管理
解决什么：把高质量提示词做成可复用模板，分类管理、一键套用，减少重复劳动。
  • 在编辑器顶部预设条，点分类胶囊即可套用到中文 / 英文栏（左键填中文、右键填英文）。
  • 点「➕」新增自定义分类与模板。
  • 点「🔄」一键还原默认预设库。
  • 内置「画质构图 / 镜头视角 / 运镜方式 / 风格滤镜」四大默认分组。
模板套用自定义分类默认库
7

⭐ 智能分镜

AI 编剧 · 可编辑分镜表 · 一致性锁定
解决什么：AI 自动写分镜 + 类 Excel 可编辑表格，并自动保证角色 / 场景 / 道具跨镜头一致性，避免成片"换脸漂移"。
  • 导航点「智能分镜」，输入创意点「一键生成」。
  • 系统并行提取角色与场景，再生成每镜的镜头编号、景别、运镜、台词旁白、时长、备注。
  • 表格可直接编辑任意单元格，拖拽调整顺序。
  • 上传参考图资产：角色图控身份 / 服装，场景图控环境，道具图控道具，防漂移。
  • 导出为 .xlsx（默认）或 .json 交付拍摄 / 制作。
AI 编剧分镜表编辑一致性锁定参考图资产
8

⭐ 视频提示词生成

文案 → 即梦/可灵/Seedance/Veo 镜头级提示词
解决什么：将一段文案自动拆解为可直接粘贴到各视频平台的镜头级提示词，省去逐条手写。
  • 导航点「视频提示词」，左面板选参数、右面板粘贴文案。
  • 点「生成」，支持多维度组合：输出格式、时间格式、内容密度、模型适配、画幅（16:9 / 9:16 / 1:1 / 21:9）、语言、控制强度。
  • 可「导入文本」批量读入，或点「查看示例」参考格式。
  • 生成后「复制输出」或「导出」为 .txt / .docx。
  • 用「模型连通性检测」确认当前 API 配置可用。
文案拆镜头多平台适配画幅/密度可调一键导出
9

视频提示词导出工作台

批量管理 · 多版本对比 · 一键交付
解决什么：生成了大量视频提示词后怎么管？导出工作台提供集中管理、多版本对比、按场景筛选的能力，让交付更专业。
  • 导航点「视频提示词导出」，查看所有已生成的提示词集合。
  • 按输出平台（即梦/可灵/Seedance/Veo/Runway）、画幅、时间格式筛选和排序。
  • 支持多版本并排对比：同一文案在不同参数下的输出差异一目了然。
  • 选中需要的条目，批量导出为 .txt / .docx 或直接复制到剪贴板。
  • 导出时可选是否包含镜头编号、景别、运镜等结构化标注。
批量管理版本对比多平台筛选结构化导出
10

多格式导入导出

JSON / Excel / SRT / TXT / DOCX 全覆盖
解决什么：你的数据不应该被锁在一个软件里。KuAi 支持主流格式双向流转，方便对接其他工具和交付给团队。
  • 项目导出：工作台全部卡片和分组 → .json 完整备份，可跨电脑恢复。
  • 分镜导出：分镜表 → .xlsx（Excel 可继续编辑）或 .json 给后期/拍摄使用。
  • 提示词导出：生成的视频提示词 → .txt / .docx 直接交付。
  • SRT 字幕导出：分镜台词旁白 → .srt 字幕文件，直接导入剪辑软件。
  • 批量导入：支持从文件夹 / 剪贴板 / 拖拽批量导入提示词条目或参考图。
JSONExcelSRT 字幕DOCX
11

清理工具

脏提示词清洗 · 旧稿自动归档
解决什么：把网上复制来的脏提示词洗干净，并把项目里的旧稿自动归档，保持工作区整洁。
  • 菜单「工具 → 旧稿归档」打开归档窗口，扫描项目文件夹。
  • 把"定稿"里的旧版本自动移入"初稿"箱，只保留最新一版。
  • 通过「清洗卡片设置」配置去重 / 去噪 / 排序 / 格式化规则。
去重去噪旧稿归档规则可配

全局捕获

截图 / 抓字 → 灵感库 / 提示词
解决什么：随手把屏幕上的文字或图片抓进灵感库 / 提示词，贴合创作者随时收集素材的习惯。
  • 开启全局捕获，用截屏框选区域（带放大镜 / 工具栏）。
  • 抓取的文字直接进提示词，图片直接进视觉灵感库。

快捷键 / 工具栏自定义

热键重映射 · 配置导入导出
解决什么：按个人习惯自定义操作与热键，并把配置随身带走。
  • 菜单「视图 / 工具」打开快捷键设置窗口，修改任意热键。
  • 工具栏编辑模式下改名 / 改色 / 拖动排列。
  • 通过「导出 / 导入配置」把设置存为 .json，换电脑也能复用。
热键自定义配置迁移工具栏编辑
14

自动保存 / 历史快照

每 2 分钟存盘 · 50 个快照可回退
解决什么：防丢稿，可随时回退到任意历史版本。
  • 无需手动——每 2 分钟定时保存，生成操作后会自动存盘。
  • 系统保留最多 50 个历史快照（去重、原子写入防损坏）。
  • 在视频 / 分镜模块点「历史」按钮，浏览并恢复任意快照。
防丢稿50 快照一键回退
15

崩溃保护 / 恢复

黑匣子日志 · 崩溃前备份一键恢复
解决什么：出错不闪退，留下日志黑匣子，并能从崩溃前的备份恢复工作。
  • 启动时自动检测崩溃备份，弹窗询问「恢复 / 丢弃」。
  • 日常异常写入 crash_logs/*.log 并弹窗提示。
  • 主数据另存 软件配置数据(请勿删除).json.crash_backup.json 作为兜底。
