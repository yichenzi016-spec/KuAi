把软件安装包放到这个目录下，命名为：

    kuai-v1.0-setup.exe

首页「立即下载」按钮会自动指向它（index.html 里的链接：downloads/kuai-v1.0-setup.exe）。

如果你还没有打包安装包：
- 源码在 E:\AI\AI提示词工具最新软件架构\
- 可用 PyInstaller 打包（需先 pip install pyinstaller，并在 app/ 下准备好依赖与 2.ico 图标）
- 也可以直接把整个源码目录压缩成 zip 供用户下载（命名为 kuai-v1.0-source.zip，并相应修改 index.html 的下载链接）

本目录仅用于存放可下载文件，不要放其他无关内容。
